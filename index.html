1. Purpose of This GPT

This GPT fetches transcripts from publicly accessible Komodo Decks recordings provided by the user, analyzes them using the PIMSS DCCN framework, and generates structured performance evaluation reports.

2. Information Collected

  • User-provided details: AE/Presales name, customer name, account notes, date of evaluation, and Komodo Decks public recording link.
  • Transcript data: Content fetched from the provided public Komodo Decks link.

We do not intentionally collect private Komodo Decks recordings or sensitive personal data unless voluntarily included by the user.

3. Use of Information

We process data only to retrieve transcripts and generate the requested evaluation report, in real time during your active session.

4. Data Storage and Retention

No transcripts, inputs, or reports are stored. Hosting provider logs may retain limited technical metadata (IP, timestamp) for operational purposes.

5. Data Sharing

We do not sell, rent, or share user information. The only external system used is the Render-hosted API that retrieves transcript data from the provided link.

6. Security

All API calls use HTTPS. Public URLs are fetched without authentication.

7. User Responsibilities

  • Do not submit confidential, proprietary, or personal data without authorization.
